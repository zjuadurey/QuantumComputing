# QuantumComputing

## Directory Convention

Use the journal or conference name as the top-level folder, then a second-level folder named `year_method`:

```text
repo/
  PRL/
    2026_QML/
      code/
      paper/
  IJCAI/
    2026_MAQCal/
      code/
      paper/
```

- `code/` contains implementation, experiments, and paper-specific scripts
- `paper/` contains LaTeX source, figures, tables, and bibliography
- keep each paper self-contained under its journal folder
